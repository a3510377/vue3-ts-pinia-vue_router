# vue3-ts-pinia-vue_router

A development template for vue3 + vue-router + pinia + typescript

## init

```shell
yarn install
```

## ⚙️ Development

```shell
yarn dev
```

### Build

```shell
yarn build
```

## ️ Technologies

- [Vue](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Pinia](https://pinia.vuejs.org/)
- [Typescript](https://www.typescriptlang.org/)
